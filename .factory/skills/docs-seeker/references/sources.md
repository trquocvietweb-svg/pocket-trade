# Documentation Sources Reference

Comprehensive guide to finding documentation across ecosystems and platforms.

## context7.com (Primary Source)

**Always try first** - Aggregates llms.txt for most popular libraries.

### URL Patterns

```
GitHub repos:  https://context7.com/{org}/{repo}/llms.txt
Websites:      https://context7.com/websites/{normalized-path}/llms.txt
Topic filter:  https://context7.com/{path}/llms.txt?topic={query}
Full content:  https://context7.com/{path}/llms-full.txt
```

## Documentation Hosting Platforms

### Docusaurus (React ecosystem)
```
Pattern: {project}.io, docs.{project}.com
Examples:
- docusaurus.io
- react.dev
- reactnative.dev
```

### VitePress (Vue ecosystem)
```
Pattern: {project}.dev, {project}.vuejs.org
Examples:
- vuejs.org
- vitejs.dev
- pinia.vuejs.org
```

### Mintlify (Modern startups)
```
Pattern: docs.{company}.com
Features: Clean design, AI search
Examples:
- docs.resend.com
- docs.trigger.dev
```

### GitBook
```
Pattern: {org}.gitbook.io/{project}
Examples:
- Company documentation
- API references
```

### Read the Docs (Python ecosystem)
```
Pattern: {project}.readthedocs.io
Examples:
- requests.readthedocs.io
- django.readthedocs.io
```

### docs.rs (Rust ecosystem)
```
Pattern: docs.rs/{crate}
Auto-generated from Rust doc comments
```

### pkg.go.dev (Go ecosystem)
```
Pattern: pkg.go.dev/{module}
Auto-generated Go documentation
```

## Package Registries

### npm (JavaScript/TypeScript)
```
URL: npmjs.com/package/{name}
Info: Description, homepage, repository, versions
```

### PyPI (Python)
```
URL: pypi.org/project/{name}
Info: Description, docs link, homepage, versions
```

### crates.io (Rust)
```
URL: crates.io/crates/{name}
Info: Description, docs.rs link, repository
```

### Packagist (PHP)
```
URL: packagist.org/packages/{vendor}/{name}
Info: Description, homepage, repository
```

### RubyGems (Ruby)
```
URL: rubygems.org/gems/{name}
Info: Description, homepage, documentation
```

### Maven Central (Java)
```
URL: search.maven.org/artifact/{group}/{artifact}
Info: Versions, dependencies, repository
```

## GitHub Raw Content

Convert GitHub blob URLs to raw for direct fetch:

```
Blob:  github.com/{org}/{repo}/blob/{branch}/{path}
Raw:   raw.githubusercontent.com/{org}/{repo}/{branch}/{path}

Example:
Blob:  github.com/vercel/next.js/blob/main/README.md
Raw:   raw.githubusercontent.com/vercel/next.js/main/README.md
```

### Common Documentation Paths in Repos

```
/README.md           - Project overview
/docs/               - Documentation directory
/documentation/      - Alternative docs directory
/examples/           - Code examples
/CONTRIBUTING.md     - Contribution guide
/CHANGELOG.md        - Version history
/API.md              - API documentation
/.github/            - GitHub-specific docs
```

## Search Patterns

### Finding llms.txt
```
WebSearch: "{library} llms.txt"
WebSearch: "{library} llms.txt site:{domain}"
```

### Finding Official Repository
```
WebSearch: "{library} github official"
WebSearch: "{library} source code repository"
```

### Finding Documentation
```
WebSearch: "{library} documentation official"
WebSearch: "{library} docs getting started"
WebSearch: "{library} API reference"
```

### Domain Filtering
```
WebSearch: "{library} docs site:github.com"
WebSearch: "{library} tutorial site:dev.to"
WebSearch: "{library} guide site:medium.com"
```
